Conceitos: Categorias e Extração

                

                
            

            

                
                

                    

                         Categoria (Grupo de Documentos)
                    

                    

                        Uma categoria agrupa documentos pelo código TJ-MS (ex: 500, 510, 9500).
                        Define apenas a estrutura de extração (perguntas e formato JSON).
                    

                    

                        Exemplo: Categoria "Petições" com códigos 500, 510, 9500, 8320, 8326
                    

                


                
                

                    

                         Perguntas de Extração
                    

                    

                        As perguntas definem o que será extraído dos documentos.
                        Podem ter dependências condicionais (só são feitas se outra pergunta tiver certo valor).
                    

                    

                        Dica: Use o botão "Reordenar com IA" para organizar as perguntas automaticamente.
                    

                


                
                

                    

                         Variáveis
                    

                    

                        As variáveis são geradas a partir das perguntas.
                        Cada variável pode ter sua própria fonte de verdade.
                    

                    

                        Acesse: Admin de Variáveis para configurar fontes de verdade.
                    

                


                
                

                    

                         Fonte de Verdade (Variáveis)
                    

                    

                        A fonte de verdade é um conceito exclusivo das variáveis.
                        Define DE QUAL documento específico a variável será extraída.
                    

                    

                        Exemplo: Extrair "medicamento" apenas do "parecer NAT", não de outros documentos.
                    

                


                
                

                    

                         Ordenação de Perguntas
                    

                    

                        Perguntas condicionais devem ficar logo abaixo de suas âncoras.
                        Use drag-and-drop ou "Reordenar com IA" para organizar.
                    

                    

                        Regra: Pergunta âncora → Suas condicionais → Próxima âncora → Suas condicionais...
